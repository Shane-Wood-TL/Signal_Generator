Detailed Description

ButtonDriver class determines if there is an input on the button present in the system (This system uses one button). Each button uses one GPIO input pin with a pull-up resistor which is designated by the constructor, a GPIO name (A, H, G,...) and a pin number (1, 3, 12,...). Also requires a pointer to a semaphore (created by another class). These are all assigned in the constructor. The "UpdateButton" member function reads the designated pin for input and with a simple state machine it checks for a falling edge. If it recognizes a falling edge it will change a the "isButtonPressed" value to 1 in an inputValues struct. The UpdateButton function will only run when the semaphore value is "true".
