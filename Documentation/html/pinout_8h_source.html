11/*
12 * FCLK is set to 72Mhz
13 * FCLK also needs to be updated in allIncludes.h
14 *
15 * Dac setup
16 * CH1 -> triggered on timer 2
17 * CH2 -> triggered on timer 6
18 * PA_4 and PA_5 need to be set up under the dac's GPIO settings in analog mode
19 *
20 * DMA setup
21 * DMA, enabled
22 * Memory to peripheral
23 * size of word, circular
24 * both channels enabled (3 and 4)
25 * DAC global interrupts for ch3 and ch4
26 *
27 * Timers
28 * 4 timers (2,6,7,16 are currently used)
29 * global interrupts for 7,16 (semaphores)
30 *
31 * timer 2, 6's ARR is set in the code
32 * timerPSC needs to be set for what matches timer 2 and 6 (this is only in allIncludes.h)
33 * more pre-scaling might be needed if the frequency needs to go lower or the clk speed has changed
34 * due to some timers only being 16 bits(notable tim6) a pre-scaling of 4 was needed to prevent overflow
35 *
36 *
37 * the values for timers are set in the ioc file
38 * timer 7 needs a period of 2ms adjusted in the ioc file (pre-scaling + arr can be used,
39 * will vary based on clock speed)
40 * At 72Mhz Arr = 5000, pre-scaling = 28
41 * timer 16 needs a period of 1s adjusted in the ioc file (same as timer 7)
42 * At 72Mhz Arr = 61000, pre-scaling = 1200
43 *
44 *
45 * SPI Setup
46 * 36Mbit/s (other speed work as well, although this is built for 36Mbit/s)
47 * 8 bits, Motorola format
48 * Mode: transmit only master (display provides no feedback)
49 * no interrupts or DMA used for SPI
50 *
51 *
52 * Pin Setups
53 * Output
54 * dc (B7), rst(B6)
55 *
56 * set pins as spi3 mosi
57 * mosi (B5)
58 *
59 * set pins as spi3 sclk
60 * sclk (B3)
61 *
62 * PA_4 needs to be set for DAC_OUT1 output
63 * PA_5 needs to be set for DAC_OUT2 output
64 *
65 * B4 can also be set as an output pin if the TOGGLE /WRITEON / WRITEOFF macros are being used
66 *
67 * Other pinouts + settings below #define for the
68 * display
69*/
