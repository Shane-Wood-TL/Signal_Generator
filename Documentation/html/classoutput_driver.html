void outputDriver::update ( )

update and resync the dacDrivers

Waves will get out of sync when frequencies are changed. To fix this when waves are brought back to the same frequency they are resynced back together by restarting DMA. The same thing occurs for ECHO as the timing in-between the waves needs to be exact
